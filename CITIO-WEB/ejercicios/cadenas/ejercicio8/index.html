Ejercicio 8: Comparar Frases

Pedir dos frases e indicar cual tiene mayor cantidad de caracteres

Resultado:

💡 ¿Cómo funciona este ejercicio?

📝 Comparación de Frases

  • 🔤 Ingresa dos frases diferentes
  • 📏 El programa cuenta los caracteres de cada frase
  • ⚖️ Compara las longitudes y determina cuál es mayor
  • 📊 Muestra el resultado con detalles

🧠 Lógica del Ejercicio:

let frase1 = prompt("Ingresa la primera frase");
let frase2 = prompt("Ingresa la segunda frase");
let longitud1 = frase1.length;
let longitud2 = frase2.length;

if (longitud1 > longitud2) {
    console.log("La primera frase es más larga");
} else if (longitud2 > longitud1) {
    console.log("La segunda frase es más larga");
} else {
    console.log("Ambas frases tienen la misma longitud");
}
