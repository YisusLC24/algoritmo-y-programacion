Ejercicio 9: Contar Letra Específica

Contar cuántas veces aparece una letra específica

Resultado:

💡 ¿Cómo funciona este ejercicio?

🔤 Contador de Letras

  • 📝 Ingresa un texto de cualquier longitud
  • 🔍 Especifica qué letra quieres buscar
  • ⚙️ Opción para distinguir mayúsculas/minúsculas
  • 📊 Ve el conteo y las posiciones donde aparece

🧠 Lógica del Ejercicio:

let texto = prompt("Ingresa un texto");
let letra = prompt("Ingresa la letra a buscar");
let contador = 0;

for (let i = 0; i < texto.length; i++) {
    if (texto[i].toLowerCase() === letra.toLowerCase()) {
        contador++;
    }
}

console.log(`La letra "${letra}" aparece ${contador} veces`);
