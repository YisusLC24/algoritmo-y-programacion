Ejercicio 9: Copiar Números Pares

Copiar solo los pares de los elementos de un arreglo en otro arreglo

💡
Ingresa un arreglo para ver los números pares copiados

💡 ¿Cómo funciona este ejercicio?

¿Qué es copiar números pares? Es crear un nuevo arreglo que contenga únicamente los números pares del arreglo original.

Proceso:

  • 🔢 Recorrer cada elemento del arreglo original
  • ✅ Verificar si el número es par (resto de división entre 2 es 0)
  • 📋 Si es par, copiarlo al nuevo arreglo
  • 📊 Mostrar ambos arreglos: original y pares

Ejemplo: Los números pares de [1,2,3,4,5,6] son: [2,4,6]

🧠 Lógica del Ejercicio:

let arregloOriginal = [1, 2, 3, 4, 5, 6, 7, 8];
let arregloPares = [];

for (let i = 0; i < arregloOriginal.length; i++) {
    if (arregloOriginal[i] % 2 === 0) {
        arregloPares.push(arregloOriginal[i]);
    }
}

console.log("Arreglo original:", arregloOriginal);
console.log("Números pares:", arregloPares);
