Ejercicio 3: Contar Mayores de Edad

Dado un arreglo de edades, contar cuántos son mayores de edad (18 años o más).

💡 ¿Cómo funciona este ejercicio?

¿Qué es mayor de edad? En la mayoría de países, una persona es mayor de edad a partir de los 18 años.

📝 Proceso:

  • 🔤 Convertir el texto en un arreglo de números
  • 🔍 Revisar cada edad del arreglo
  • ✅ Contar las edades >= 18
  • 📊 Mostrar el resultado y estadísticas

Ejemplo: Edades [15, 22, 17, 25, 30, 16, 19] → Mayores de edad: 4 (22, 25, 30, 19)

🧠 Lógica del Ejercicio:

let edades = [15, 22, 17, 25, 30, 16, 19];
let mayoresDeEdad = 0;

for (let i = 0; i < edades.length; i++) {
    if (edades[i] >= 18) {
        mayoresDeEdad++;
    }
}

console.log("Mayores de edad: " + mayoresDeEdad);
