Ejercicio 6: Saludo Personalizado por Género

👋 Ingresa tu nombre y género para recibir un saludo personalizado

Resultado:

👋 Ingresa tu nombre y género para recibir un saludo personalizado

🤔 ¿Cómo funciona este ejercicio?

👨 Masculino

"Bienvenido"

Saludo dirigido a personas de género masculino
👩 Femenino

"Bienvenida"

Saludo dirigido a personas de género femenino
👤 Otro

"Bienvenido/a"

Saludo inclusivo para todas las personas

🧠 Lógica del Ejercicio:

if (genero === "masculino") {
    saludo = "¡Bienvenido " + nombre + "!";
    emoji = "👨";
} else if (genero === "femenino") {
    saludo = "¡Bienvenida " + nombre + "!";
    emoji = "👩";
} else {
    saludo = "¡Bienvenido/a " + nombre + "!";
    emoji = "👋";
}
