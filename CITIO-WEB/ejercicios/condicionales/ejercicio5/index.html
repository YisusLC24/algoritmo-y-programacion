Ejercicio 5: Clasificación de Temperatura

🌡️ Ingresa una temperatura para ver la clasificación

Resultado:

🌡️ Ingresa una temperatura para ver la clasificación

🤔 ¿Cómo funciona este ejercicio?

🥶 Frío

Menos de 15°C

Temperaturas bajas, necesitas abrigo
🌤️ Templado

15°C - 25°C

Temperatura agradable y cómoda
🔥 Calor

Más de 25°C

Temperaturas altas, mantente hidratado

🧠 Lógica del Ejercicio:

if (temperatura < 15) {
    clasificacion = "Frío 🥶";
    descripcion = "Hace frío, abrígate bien";
} else if (temperatura >= 15 && temperatura <= 25) {
    clasificacion = "Templado 🌤️";
    descripcion = "Temperatura agradable";
} else {
    clasificacion = "Calor 🔥";
    descripcion = "Hace calor, mantente hidratado";
}
