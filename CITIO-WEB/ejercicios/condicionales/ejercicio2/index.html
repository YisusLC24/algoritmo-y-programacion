Ejercicio 2: Comparar Cuatro Números

Pedir cuatro números y mostrar cuál es mayor, y si son iguales decirlo.

Resultado:

👆 Ingresa cuatro números para ver la comparación

💡 ¿Cómo funciona este ejercicio?

🔢 Comparaciones Múltiples

Aprende a comparar varios valores usando operadores relacionales como >, <, >=, <=, ==

Ideal para entender comparaciones complejas
⚖️ Estructuras Condicionales

Domina el uso de if, else if y else para manejar múltiples condiciones

Fundamental para la lógica de programación
🎯 Detección de Igualdades

Identifica cuando varios números son iguales y maneja estos casos especiales

Perfecto para casos edge en programación

📝 Lógica del Ejercicio:

if (num1 == num2 && num1 == num3 && num1 == num4) {
    comparacion = "Todos los números son iguales";
} else if (num1 >= num2 && num1 >= num3 && num1 >= num4) {
    comparacion = "El mayor es: " + num1;
} else if (num2 >= num1 && num2 >= num3 && num2 >= num4) {
    comparacion = "El mayor es: " + num2;
} else if (num3 >= num1 && num3 >= num2 && num3 >= num4) {
    comparacion = "El mayor es: " + num3;
} else {
    comparacion = "El mayor es: " + num4;
}
