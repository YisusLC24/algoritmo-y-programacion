Ejercicio 2: Tabla de Multiplicar

Pedir un número y mostrar su tabla de multiplicar del 1 al 10 usando un ciclo for.

Resultado:

🔢 Ingresa un número para ver su tabla de multiplicar

💡 ¿Cómo funciona este ejercicio?

📝 Niño (0-12)

🧑 Adolescente (13-17)

👨 Adulto (18-59)

👴 Adulto Mayor (60+)

💡 Lógica del Ejercicio:

let numero = parseInt(prompt("Ingresa un número"));
for (let i = 1; i <= 10; i++) {
    let resultado = numero * i;
    console.log(numero + " x " + i + " = " + resultado);
}

Explicación:

  • Input: Recibimos un número del usuario
  • Ciclo for: i va del 1 al 10
  • Operación: numero * i para cada iteración
  • Output: Mostramos "numero x i = resultado"
