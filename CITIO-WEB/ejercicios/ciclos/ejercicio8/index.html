Ejercicio 8: Contador de Nombres hasta "Messi"

Pedir nombres hasta que el usuario escriba "Messi". Al final presente cuántos nombres ingresó con while

Nombres ingresados: 0

💡 Escribe nombres y presiona "Agregar Nombre" o Enter. Escribe "Messi" para terminar.

💡 ¿Cómo funciona este ejercicio?

Este ejercicio usa un ciclo while que continúa ejecutándose hasta que se cumple una condición específica.

Proceso:

  • 🔄 Usar ciclo while que continúe hasta que el usuario escriba "Messi"
  • 📝 Pedir nombres uno por uno
  • 🔢 Contar cada nombre ingresado
  • 🛑 Parar cuando aparezca "Messi"
  • 📊 Mostrar el total de nombres ingresados

🔧 Lógica del Ejercicio:

let contador = 0;
let nombre = "";
let nombres = [];

while (nombre.toLowerCase() !== "messi") {
    nombre = prompt("Ingresa un nombre:");
    if (nombre.toLowerCase() !== "messi") {
        nombres.push(nombre);
        contador++;
    }
}

alert("Total de nombres: " + contador);

Ejemplo: Si ingresas: Juan, María, Pedro, Messi → Total: 3 nombres
